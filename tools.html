IP 查询

输入 IPv4/IPv6 地址进行查询，或点击“检测我的 IP”获取当前公网信息。默认查询使用公共地址数据库 ip-api.com（街区级精度），如果该服务拒绝会自动退回到 ipapi.co。

等待查询 — 输入 IP 或使用“检测我的 IP”。

网站连通性检测

请输入要检测的 URL 或域名（支持自动补全 https://）。工具将尝试以 HEAD 请求检测响应码和响应时延，若被阻止会自动退回到 GET。注意：由于浏览器跨域限制，部分站点可能无法直接返回状态（提示将说明原因）。

等待检测。

MC 服务器检测

检测 Minecraft Java/Bedrock 服务器状态（使用 mcsrvstat API）。推荐服务器： luminya.cc/lumiser（LumiSer 星光智服）。

等待检测。

Base64 编解码

支持 UTF-8 文本。选择“编码”或“解码”，并可启用 URL-safe（替换 +/ 为 -_）。

等待操作。
