这是一个用于展示个人主页的静态网页项目，核心文件为 `index.html`，以 HTML 和内联 CSS 编写。

## 特性

- **毛玻璃风格界面**：顶部导航、卡片和链接区均使用透明度和模糊效果，配合浅蓝白配色。
- **响应式设计**：支持移动设备视图，使用媒体查询调整字体和布局。
- **本地资源**：头像和背景图像引用 `/img` 目录下的资源。
- **Font Awesome 图标**：通过 CDN 引入丰富图标用于装饰。
- **个人信息展示**：包含昵称、简介、链接按钮网格和页脚。

## 目录结构

```
index.html      # 个人主页，展示简介与链接
contact.html    # 联系页面，内含表单与联系方式
history.html    # 网站搭建历程和时间线展示
lumiser.html    # LumiSer 服务器介绍页
whois.html      # 域名收藏馆与实时 WHOIS 查询
img/            # 存放背景图、头像以及其他静态资源
js/             # 可放置将来需要的脚本文件（当前为空）
LICENSE         # 许可文件
readme.md       # 项目说明
```

## 使用说明

1. 将项目文件放置于 Web 服务器根目录或本地文件夹。
2. `/img` 目录应包含必要图片（例如 `bg.webp`、`logo.webp` 等），否则背景及头像可能无法显示。
3. 打开任意 `.html` 页面（如 `index.html`、`contact.html` 等）即可预览对应内容；各页面之间通过顶栏和链接相互导航。
4. `contact.html` 需要保持网络访问权限以加载 Quill 编辑器、EmailJS 相关 CDN 脚本。若需要离线支持，可将这些库下载并放在 `js/` 目录。
5. 若拓展功能或添加自定义 JavaScript，可在 `js/` 目录新建文件并通过 `<script>` 引入。

## 自定义

可根据需要更改文本内容、链接、图像路径和配色方案。

### 样式调整

内联 `<style>` 块中包含全局重置、毛玻璃卡片、头像容器、按钮、响应式媒体查询等。想要修改配色、圆角、阴影或布局，可直接编辑该部分代码。

### 资源管理

- 背景图：`/img/bg.webp`，建议使用大尺寸、压缩后 WebP 格式。
- 头像：`/img/logo.webp`；容器使用圆形裁剪，若不想显示图像可将该元素替换为其他内容。
- 其他图标全由 Font Awesome 提供，可在 `<i>` 标签中更换类名实现不同主题。

## 开发说明

这是一个纯前端静态页面，无需构建工具或依赖。只要浏览器支持现代 CSS 即可，开发者可通过本地 HTTP 服务器（如 `live-server`、`python -m http.server`）预览。

### 浏览器兼容性

页面使用 CSS 自定义属性较少，大多数现代浏览器（Chrome、Edge、Firefox、Safari）均可正常渲染。旧版 IE8/9 不支持 `backdrop-filter` 和 `object-fit`，在这些浏览器下毛玻璃效果会退化为半透明背景。
